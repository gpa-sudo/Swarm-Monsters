# 🕷️ Swarm Monsters

A browser-based arcade survival game built with HTML5 Canvas.

## 🕹️ How to Play

- Move your cursor to control your character
- **Spider mode** — shoot webs to trap and kill enemies
- **Scorpion mode** — crush enemies on contact using heat charge
- Survive waves of increasingly dangerous swarm monsters
- Pick upgrades between waves to build your build

## ⌨️ Controls

| Input | Action |
|-------|--------|
| Mouse Move | Move character |
| Left Click | Use ability (web shot / charge) |

## 🧬 Game Modes

- **Spider** — Tactical web-trapping playstyle with combo kills
- **Scorpion** — Aggressive heat-based melee playstyle

## ⚡ Features

- Wave-based survival with escalating difficulty
- Upgrade/perk system between waves
- Combo multiplier system
- Boss enemy (Subject 0)
- Screen shake, particles, and visual effects
- Fully self-contained — no dependencies, runs in any browser

## 🛠️ Tech Stack

- Vanilla HTML5 / CSS3 / JavaScript
- HTML5 Canvas (2D rendering)
- No frameworks or libraries

## 🚀 Run Locally

Just open `index.html` in any modern browser — no server needed.
